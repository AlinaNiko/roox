Способы идентификации и идентификаторы

  • Гибкая настройка аутентификации – поддержка разных идентификаторов

    RooX UIDM упрощает процесс аутентификации пользователей, поддерживая различные идентификаторы абонентов. Это позволяет настроить процедуру аутентификации, добавив в нее новые средства: например, аутентификацию по номеру телефона, IP-адресу, MSISDN, при помощи одноразового пароля по SMS / EMail, RADIUS.

  • Автоматическая ассоциация для доверенных сервисов

    Для доверенных сервисов (например, два портала оператора) автоматическая ассоциация учетных записей, имеющих какой-либо общий параметр (например, мобильный номер), осуществляется автоматически при первом переходе абонента между данными сервисами.

  • Ассоциация при отсутствии общих идентификаторов

    Для сервисов, не имеющих общих идентификаторов (например, для GSM или ШПД), либо для сервисов, не имеющих доверительных отношений (например, портал оператора и социальная сеть), предоставляется возможность ручной ассоциации учетных записей через диалог с абонентом. Также в случае ШПД – возможна аутентификация абонента и чужого оператора с использованием MSISDN.

  • Многоуровневый доступ, определяемый точкой аутентификации

    RooX позволяет оператору внедрять новые сложные модели услуг путем сопоставления услуг с уровнем аутентификации абонента, определяемым его / ее точкой аутентификации.

    Содержимое портала, доступное подписчику, будет различаться для разных уровней доступа.

    • Самый низкий уровень может быть назначен анонимному подписчику.
    • Более высокие уровни будут связаны с входом через учетную запись социальной сети или с использованием IP-адреса, имени пользователя / пароля, устройства подписчика или одноразового пароля по SMS.

    Веб-приложения могут запрашивать обновления уровня - и RooX UIDM будет взаимодействовать с подписчиком для обработки этих запросов.
